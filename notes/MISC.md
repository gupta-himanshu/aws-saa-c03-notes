# Miscellaneous AWS Notes

- AWS VM Import/Export can migrate virtual machines.
- Trusted Advisor monitors service limits and resource usage.
- Security groups restrict application-to-application communication.
- S3 Transfer Acceleration improves upload speed only.
- CloudFront improves download speed from S3.
- Gateway Endpoints improve secure access to S3 and DynamoDB.
- Route 53 health checks can monitor external endpoints.
